# Configuration Reference

`bifrost` and `bifrost-tui` load a JSON config file via `--config <path>`.

## Config Schema

Example:

```json
{
  "group_path": "./data/group.json",
  "share_path": "./data/share-alice.json",
  "state_path": "./data/state-alice.json",
  "relays": ["ws://127.0.0.1:8194"],
  "peers": [
    {
      "pubkey": "<peer-member-pubkey-hex>",
      "policy": {
        "block_all": false,
        "request": { "echo": true, "ping": true, "onboard": true, "sign": true, "ecdh": true },
        "respond": { "echo": true, "ping": true, "onboard": true, "sign": true, "ecdh": true }
      }
    }
  ],
  "options": {
    "sign_timeout_secs": 30,
    "ecdh_timeout_secs": 30,
    "ping_timeout_secs": 15,
    "onboard_timeout_secs": 30,
    "request_ttl_secs": 300,
    "max_future_skew_secs": 30,
    "request_cache_limit": 2048,
    "state_save_interval_secs": 30,
    "event_kind": 20000,
    "peer_selection_strategy": "deterministic_sorted",
    "router_expire_tick_ms": 1000,
    "router_relay_backoff_ms": 50,
    "router_command_queue_capacity": 128,
    "router_inbound_queue_capacity": 4096,
    "router_outbound_queue_capacity": 1024,
    "router_command_overflow_policy": "fail",
    "router_inbound_overflow_policy": "drop_oldest",
    "router_outbound_overflow_policy": "fail",
    "router_inbound_dedupe_cache_limit": 16384
  }
}
```

## Field Semantics

- `group_path`: group package JSON path.
- `share_path`: local share package JSON path.
- `state_path`: signer state snapshot path.
- `relays`: relay URL list. Must be non-empty.
- `peers`: allowed peer pubkeys and per-peer policy.
- `options`: runtime timeout/event settings.
  - `request_ttl_secs`: max allowed age for inbound `request_id` values before rejection.
  - `max_future_skew_secs`: max allowed future timestamp skew for inbound `request_id` values.
  - `request_cache_limit`: replay-cache entry cap before oldest entries are evicted.
  - `router_expire_tick_ms`: bridge pending-round expiry sweep interval in milliseconds.
  - `router_relay_backoff_ms`: bridge backoff delay after relay read errors in milliseconds.
  - `router_command_queue_capacity`: max queued local commands before overflow policy applies.
  - `router_inbound_queue_capacity`: max buffered relay events before overflow policy applies.
  - `router_outbound_queue_capacity`: max buffered outbound events before overflow policy applies.
  - `router_command_overflow_policy`: `fail` or `drop_oldest`.
  - `router_inbound_overflow_policy`: `fail` or `drop_oldest`.
  - `router_outbound_overflow_policy`: `fail` or `drop_oldest`.
  - `router_inbound_dedupe_cache_limit`: max inbound event IDs kept for duplicate suppression.
  - `peer_selection_strategy`:
    - `deterministic_sorted` (default): first `N` eligible peers in sorted pubkey order.
    - `random`: random sample of `N` eligible peers.

## Defaults

When generated by `bifrost-devtools keygen`:
- `event_kind`: `20000`
- `peer_selection_strategy`: `deterministic_sorted`
- `sign_timeout_secs`: `30`
- `ecdh_timeout_secs`: `30`
- `ping_timeout_secs`: `15`
- `onboard_timeout_secs`: `30`
- `request_ttl_secs`: `300`
- `max_future_skew_secs`: `30`
- `request_cache_limit`: `2048`
- `state_save_interval_secs`: `30`
- `router_expire_tick_ms`: `1000`
- `router_relay_backoff_ms`: `50`
- `router_command_queue_capacity`: `128`
- `router_inbound_queue_capacity`: `4096`
- `router_outbound_queue_capacity`: `1024`
- `router_command_overflow_policy`: `fail`
- `router_inbound_overflow_policy`: `drop_oldest`
- `router_outbound_overflow_policy`: `fail`
- `router_inbound_dedupe_cache_limit`: `16384`

## Validation Notes

- Peer pubkeys must be compressed secp256k1 hex.
- Member indexes and pubkeys must match the configured group package.
- Event kind must match across participating peers.
